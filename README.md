## 📋 Overview

This repository contains the official user manual for **UniShare** — a barter trading system that allows users to exchange goods and services without monetary transactions. The manual covers system usage, features, and workflows for end users.

> This is a documentation-only repository. For the UniShare application source code, refer to the main UniShare repository.

## 📘 About UniShare

**UniShare** is a barter trading system designed to connect users who want to trade goods or services directly — no money involved. Built as an academic project at PUP-SJ.
